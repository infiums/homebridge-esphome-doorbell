# Homebridge ESPHome Doorbell

This is a Homebridge platform plugin to be used with the ESP Domofon.
It is a work in progress.

## Getting Started

```json
{
    "platform": "esphome-domofon",
    "devices": [
        {
            "host": "domofon.local",
            "password": "",
            "name": "Domofon",
            "port": 9001
        }
    ]
}
```
